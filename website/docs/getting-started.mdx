## Hello World Example
<Tabs
  groupId="providerx-helloworld-example"
  defaultValue="react-providerx-helloworld-example"
  values={[
    { label: 'React', value: 'react-providerx-helloworld-example', },
  ]}
>
<TabItem value="react-providerx-helloworld-example">

```typescript
import { from } from 'rxjs'
import { useProvider } from 'react-providerx'

const messageProvider = new ObservableProvider(() => {
  return of('hello world')
})

function App() {
  const { data } = useProvider(messageProvider)
  return (
    <div>
      {data}
    </div>
  )
}
```
